PocketLadio::Deux
---


【主な機能】
PocketLadio::DeuxはMicrosoft Windows Mobile 5.0 Pocket PC上で動作する「ねとらじ」
「Podcast」「SHOUTcast」「Icecast」のヘッドラインをインターネットより取得、表示
するツールです。

ねとらじ
http://live.ladio.livedoor.com/

SHOUTcast
http://www.shoutcast.com/

Icecast
http://dir.xiph.org/index.php


【必要環境】
Microsoft Windows Mobile 5.0 Pocket PC
.NET Compact Framework 2.0
TCPMP（推奨）

.NET Compact Frameworkについては
http://www.microsoft.com/japan/msdn/vstudio/device/netcf/
を参照してください。


【インストール・アップデート】
ZIPファイルを解凍し、PocketLadioDeuxab.CABをインストールしてください。


【アンインストール】
[スタート]-[設定]-[システム]タブ-[プログラムの削除]から[garage.uraroji.com 
PocketLadio Deux]を選択して削除してください。


【使い方】
・PocketLadio::Deuxの使い方
Updateボタンを押し、配信中のヘッドラインを取得します。
Playボタンで選択した番組を再生します。
Filterチェックボックスであらかじめ設定したフィルター条件に一致する番組のみを表
示します。

・PocketLadio::Deuxの設定
トップ画面から[メニュー]->[設定]を選択するとPocketLadio::Deuxの設定を行うことが
できます。
[メディアプレーヤーのファイルパス]は番組を再生するためのメディアプレーヤーのファ
イルパスを設定します。
メディアプレーヤーにはTCPMP（http://tcpmp.corecodec.org/about）をおすすめしま
す。
[ブラウザのファイルパス]は番組のウェブサイトにアクセスするためのブラウザのファ
イルパスを設定します。

・ヘッドラインの追加
トップ画面から[メニュー]->[ヘッドラインの追加・削除]を選択すると取得したいヘッ
ドラインの追加をすることができます。
ヘッドライン名と種類を選択し[追加]ボタンでヘッドラインを追加することができます。

・ねとらじ設定
トップ画面から[メニュー]->[ヘッドラインの設定]->[(ヘッドライン名) - ねとらじ]を
選択すると、ねとらじの設定を行うことができます。
[表示フォーマット]はトップ画面での番組の表示方法を設定します。

・ねとらじ設定 表示フォーマット
[[NAME]]    - ヘッドライン名
[[GENRE]]   - 番組のジャンル
[[CLN]]     - 現リスナ数
[[CLNS]]    - 延べリスナ数
[[TITLE]]   - 現在の曲名情報
[[TIMES]]   - 放送開始時間
[[BIT]]     - ビットレート
[[PLAYURL]] - 番組の放送URL

・Podcast設定
トップ画面から[メニュー]->[ヘッドラインの設定]->[(ヘッドライン名) - Podcast]を
選択すると、Podcastの設定を行うことができます。
[表示フォーマット]はトップ画面での番組の表示方法を設定します。

・Podcast設定 表示フォーマット
[[TITLE]]       - 番組のタイトル
[[DESCRIPTION]] - 番組の詳細
[[CATEGORY]]    - 番組のカテゴリ
[[AUTHOR]]      - 番組の著者

・SHOUTcast設定
トップ画面から[メニュー]->[ヘッドラインの設定]->[(ヘッドライン名) - SHOUTcast]
を選択すると、SHOUTcastの設定を行うことができます。
[表示フォーマット]はトップ画面での番組の表示方法を設定します。
[検索ワード]はSHOUTcastの番組の検索キーワードを指定します。

・SHOUTcast設定 表示フォーマット
[[TITLE]]       - 番組のタイトル
[[DESCRIPTION]] - 番組の詳細
[[LISTENER]]    - リスナ数
[[CATEGORY]]    - 番組のカテゴリ
[[BIT]]         - ビットレート

・Icecast設定
トップ画面から[メニュー]->[ヘッドラインの設定]->[(ヘッドライン名) - Icecast]を
選択すると、Icecastの設定を行うことができます。
[表示フォーマット]はトップ画面での番組の表示方法を設定します。
[検索ワード]はIcecastの番組の検索キーワードを指定します。

・Icecast設定 表示フォーマット
[[SERVERNAME]]  - サーバー名
[[SERVERTYPE]]  - ストリーミングの種類
[[GENRE]]       - 番組のジャンル
[[CURRENTSONG]] - 現在演奏中の曲
[[BIT]]         - ビットレート

用に起因する一切の損害について開発者及び関係者が責任を負わないものとします。
